# 🚢 Smart Machine Learning System with Interactive GUI (Titanic-Based)

## 📌 Overview

This project is a full-featured **Machine Learning System with a custom-built GUI** designed to simplify the entire ML workflow — from data loading to real-time prediction.

Built using the famous Titanic dataset from Kaggle as a base, the system is designed to work with **any dataset**, making it a flexible and powerful ML tool.

---

## 🚀 Key Features

### 📂 1. Dynamic Data Loading

* Load datasets directly from your local device
* Supports flexible dataset structures
* Automatically detects features and target

---

### 📊 2. Interactive Data Exploration

* Visualize data through dynamic charts
* Explore distributions and relationships
* Gain insights before modeling

---

### 🧹 3. Data Preprocessing

* Handle missing values efficiently
* Encode categorical variables
* Normalize and scale data

---

### 🤖 4. Machine Learning Models

Run multiple Supervised Learning algorithms, including:

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)

---

### 📈 5. Model Evaluation & Comparison

* Automatically evaluates all models
* Compares performance metrics
* Selects the best-performing model

---

### 🎯 6. Smart Prediction System

* Uses the best model for predictions
* Accepts user input through GUI
* Provides real-time predictions

---

### 💻 7. Custom GUI (Qt Design)

* Clean and intuitive interface
* Easy navigation between steps
* Fully interactive experience

---

## 🛠️ Tech Stack

* Python
* Pandas & NumPy
* Scikit-learn
* Matplotlib / Seaborn
* PyQt / Qt Designer

---

## 🧠 System Workflow

1. Load Dataset
2. Explore Data
3. Preprocess Data
4. Train Models
5. Evaluate Models
6. Select Best Model
7. Predict

---

## 📊 Example Use Case

Using the Titanic dataset:

* Predict whether a passenger survives or not
* Analyze important features like Age, Fare, and Pclass
* Compare different models to find the most accurate one
